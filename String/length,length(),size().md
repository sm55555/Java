# Length, Length(), size()

### 1. length
 - arrays(int[], double[], String[])

 - length는 배열의 길이를 알고자 할때 사용된다.

### 2. length()
 - String related Object(String, StringBuilder etc)

 - length()는 문자열의 길이를 알고자 할때 사용된다. 

### 3. size()
 - Collection Object(ArrayList, Set etc)

 - size()는 컬렉션프레임워크 타입의 길이를 알고자 할때 사용된다
